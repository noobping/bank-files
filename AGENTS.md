# AI Goal

The AI must follow the code guidelines in `CONTRIBUTING.md`.

When generating, changing, reviewing, or refactoring code, always check the result against `CONTRIBUTING.md`.

If a request conflicts with `CONTRIBUTING.md`, follow `CONTRIBUTING.md` and explain why.

Do not produce code that violates the project rules for Rust style, separation of concerns, GTK/libadwaita UI, dialogs, button styling, threading, file size, modules, or Flathub/GNOME expectations.
